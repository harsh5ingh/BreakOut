# BreakOut Game

A classic Break Out arcade game built with HTML5, CSS3, and JavaScript for a college mini project.

## About the Game

Break Out is a classic arcade game where the player controls a paddle at the bottom of the screen and must break all the bricks by bouncing a ball off the paddle. The game features:

- **45 colorful bricks** arranged in 5 rows
- **Lives system** - Start with 3 lives
- **Score tracking** - Earn 10 points for each brick destroyed
- **Smooth controls** - Use mouse or keyboard
- **Win/Lose conditions** - Clear all bricks to win, lose all lives to lose

## How to Play

1. Open `index.html` in a web browser
2. Press **Space** or click to start the game
3. Control the paddle using:
   - **Mouse movement** - Move mouse left/right over the canvas
   - **Arrow keys** - Use Left/Right arrow keys
4. Keep the ball in play by hitting it with the paddle
5. Break all the bricks to win!
6. Press **Space** to pause/unpause during gameplay
7. Click **Restart Game** button to start a new game

## Game Controls

- **Mouse**: Move paddle left/right
- **Arrow Keys**: Move paddle left/right
- **Space**: Start game / Pause game
- **Restart Button**: Reset and start a new game

## Features

## Installation & Running

No installation required! Simply:

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start playing!

## Game Rules

1. The ball starts in the center of the screen
2. Use the paddle to keep the ball bouncing
3. Each brick you break earns you 10 points
4. If the ball falls past the paddle, you lose a life
5. The game ends when:
   - You clear all bricks (You Win!)
   - You lose all 3 lives (Game Over!)
